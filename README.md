# E-Commerce-analytics
Project Overview

This project analyzes e-commerce sales data to identify key trends in revenue, customer behavior, and product performance. The goal is to generate actionable insights that help improve sales strategy and business decision-making.
Dataset Description

The dataset includes transactional sales data with the following key attributes:

Order ID – Unique identifier for each order

Order Date – Date of purchase

Customer ID – Unique customer identifier

Product Category – Category of the product

Product Name – Name of the product

Quantity – Units sold

Price – Price per unit

Sales – Total sales amount

Region – Sales region
Objectives

Analyze overall sales and revenue trends

Identify top-performing products and categories

Understand customer purchasing behavior

Perform time-based sales analysis
Tools & Technologies Used

Python

Pandas – Data cleaning & analysis

NumPy – Numerical computations

plotly – Data visualization




Support data-driven business decisions
